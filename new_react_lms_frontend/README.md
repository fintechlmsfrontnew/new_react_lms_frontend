# Loan CRM (Vite + React)

Vite + React + TypeScript setup. Components yahan banenge: `src/components/`.

## Commands

```bash
npm install   # pehli baar
npm run dev   # dev server (port 3000)
npm run build # production build
npm run preview # build preview
```

## Structure

```
src/
├── components/   ← yahan components banao
├── pages/        ← screens (jab zarurat ho)
├── main.tsx
├── App.tsx
└── index.css
```
